# ChatbotUI
React native expo chatbot UI. Works on the web and phone.

You can find the code in the Master branch

## To run:
1.
```javascript
cd HealthChatBot
```
2.
```javascript
npx expo start
```
Press i to open iOS simulator

Press w to open web
